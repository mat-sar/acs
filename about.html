Annotated CSS Spec

Annotated CSS Spec combines the precision of a technical specification with the approachability of a dedicated learning resource.

What is this place?

The Annotated CSS Spec website includes a specification of the CSS box model with extra explanations and examples. This additional content helps you understand various details of the spec.

The specification comes from Chapter 8 of CSS 2.2.

How do I use it?

The specification includes little speech bubbles (). Click or tap one to open a comment about the fragment which directly precedes the bubble.

The comments are written in a way which allows you to read the spec from top to bottom, or jump right to a fragment that interests you.

Why annotated specs?

Technical specifications aim to describe, with as little ambiguity as possible, all relevant details of a technology. The goal is to allow different developers to implement the technology with similar results. Specifications are often comprehensive and precise.

They are not always easy reads, though.

The idea behind adding explanations to selected fragments of a spec is to combine the precision of a technical specification with the approachability of a dedicated learning resource.
