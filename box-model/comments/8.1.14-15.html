This comment relates to the following words from Section 8.1 of CSS 2.2:
If the margin has 0 width, the margin edge is the same as the border edge. The four margin edges define the box's margin box.

A conclusion which follows from this spec fragment is that the margin edge is present in every box. This is the case even if we set margin: 0 on the box.

The margin edge, composed of the top, right, bottom and left margin edges, defines the margin box.
