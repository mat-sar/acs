This comment relates to the following words from Section 8.1 of CSS 2.2:
If the border has 0 width, the border edge is the same as the padding edge. The four border edges define the box's border box.

Every box always has four border edges (top, right, bottom, left). This is the case even if you declare border: 0 on the box.

The four border edges come together to form a border box. A box’s border box is something different from a box’s border area:

For a little more context, you can refer to the comment about the padding box: the difference between the border box and the border area is similar to that between the padding box and the padding area.
