Annotated CSS Spec ?
This comment contains general remarks on Chapter 8 from the CSS 2.2 specification.

This chapter comes from the CSS 2.2 spec, which, at the time of writing, is only a working draft. CSS 2.2 is very similar to CSS 2.1, as it only introduces small corrections and disambiguations to its predecessor.

The topic of the chapter, the box model, is also described in a separate specification called CSS Box Model Module Level 3 . As I’m writing these words, this other specification is a work in progress, much like CSS 2.2.

Instead of annotating the new level⁠-⁠3 spec, I went with the level⁠-⁠2 chapter because it contains a description of margin collapse. That is, Section 8.3 in CSS 2 is currently the only CSS specification which explains how margins collapse. In the future, though, this topic will also be covered in a separate spec.

For completeness, let me note that the W3C is doing some early work on CSS Box Model Module Level 4. If you’re not familiar with the concept of CSS levels, the W3C website has a page which explains this and other concepts related to CSS standardization. In general, the idea is that each higher level-specification builds upon the preceding lower-level specification.
